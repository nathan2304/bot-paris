🤖 Top 3 paris “value” en direct

Version prête pour GitHub Pages. Elle récupère les cotes live d’un marché d’odds, compare chaque prix d’un bookmaker au consensus du marché, puis sort les 3 meilleures opportunités sur le marché 1X2.
Conseil : commence avec EU + Meilleur bookmaker disponible.
Prêt.
Formule
Edge ≈ (1 / cote_moyenne_marché) - (1 / cote_bookmaker)
Plus l’edge est positif, plus le prix du bookmaker choisi est favorable par rapport au marché.
